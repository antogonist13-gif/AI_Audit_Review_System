## 📋 Реализованные изменения

### 1. ✅ Добавлены 3 новые колонки в config/columns.yml

Добавлены колонки для отслеживания судебных решений:
- `cancelled_decisions` - Отменены в порядке обжалования (19 случаев)
- `returned_refused` - Возвращены, получен отказ (33 случая)
- `refused_satisfaction` - Отказано в удовлетворении (62 случая)

### 2. ✅ Реализована автоматическая классификация организаций

**Файл:** `src/svk_analytics/scoring.py`

Добавлена функция `classify_org_type()` для автоматической классификации на основе названий:
- **Высшее образование** - 232 организации (университеты, академии)
- **Научные организации** - 138 организаций (институты и центры с научной направленностью)
- **Другие** - 315 организаций

Фильтр по типу организаций теперь работает корректно в дашборде.

### 3. ✅ Разделен раздел "Нарушения и меры реагирования"

#### management_actions() - 2 показателя

#### legal_responsibility_measures() - 6 показателей

## 🎯 Как использовать изменения

### Просмотр дашборда:
```bash
source .venv/bin/activate
streamlit run app.py
```

### Обновление данных:
```bash
source .venv/bin/activate
python run_analysis.py --input "data/raw/Сводный отчет по СВК.XLS" --year 2025
```

### Доступные фильтры:
- Федеральный округ
- Регион
- **Тип организации** (теперь показывает: Высшее образование, Научные организации, Другие)
- Группа риска
