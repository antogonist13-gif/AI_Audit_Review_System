## 📝 Следующие шаги для деплоя

### 1. Инициализация Git репозитория

```bash
git init
git add .
git commit -m "Initial commit - SVK Analytics Streamlit app"
```

### 2. Создание GitHub репозитория

**Вариант A: Через GitHub CLI** (если установлен `gh`):
```bash
gh repo create svk-analytics-streamlit --private --source=. --push
```

**Вариант B: Вручную через веб-интерфейс:**

1. Зайдите на https://github.com
2. Нажмите "+" → "New repository"
3. Название: `svk-analytics-streamlit`
4. Выберите Private или Public
5. НЕ добавляйте README, .gitignore, license
6. Нажмите "Create repository"
7. Скопируйте команды для существующего репозитория:

```bash
git remote add origin https://github.com/ваш-username/svk-analytics-streamlit.git
git branch -M main
git push -u origin main
```

### 3. Деплой на Streamlit Cloud

1. Зайдите на https://streamlit.io/cloud
2. Войдите через GitHub аккаунт
3. Нажмите **"New app"**
4. Настройки:
   - Repository: `svk-analytics-streamlit`
   - Branch: `main`
   - Main file path: `app.py`
   - (Optional) Advanced settings → Python version: `3.11`
5. Нажмите **"Deploy!"**

### 4. Ожидание деплоя (3-5 минут)

Вы увидите:
- Логи установки зависимостей
- Прогресс сборки
- После успеха: публичный URL

### 5. Получите ваш URL

```
https://ваш-username-svk-analytics-streamlit.streamlit.app
```

## 💡 Альтернативные варианты

### Docker (локально или на сервере):
```bash
docker build -t svk-analytics-streamlit .
docker run -p 8501:8501 svk-analytics-streamlit
```

### Локальный запуск на другом компьютере:
```bash
git clone https://github.com/ваш-username/svk-analytics-streamlit.git
cd svk-analytics-streamlit
python -m venv .venv
source .venv/bin/activate  # macOS/Linux
# или .venv\Scripts\Activate.ps1  # Windows
pip install -r requirements.txt
streamlit run app.py
```
