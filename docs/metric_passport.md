# Паспорт метрик СВК Analytics

## 1. Покрытие активных направлений

**Смысл:** показывает, включены ли в контур внутреннего контроля те направления, где у организации фактически есть деятельность.

**Активные направления:**

- финансово-хозяйственная деятельность;
- закупки;
- имущество;
- проектная деятельность.

**Формула:**

```text
Доля покрытия = количество активных направлений, покрытых СВК / количество активных направлений
```

**Интерпретация:**

- 100% — все активные направления покрыты;
- 50–75% — частичное покрытие;
- менее 50% — выраженный риск непокрытия.

## 2. Нагрузка по направлению

**Смысл:** оценивает масштаб деятельности организации по каждому направлению относительно других организаций в отчете.

Нагрузка считается не по абсолютному нормативному порогу, а через перцентиль внутри массива организаций.

```text
0 — нет активности
1 — активность есть, но не выше 75-го перцентиля
2 — выше 75-го, но не выше 90-го перцентиля
3 — выше 90-го, но не выше 95-го перцентиля
4 — выше 95-го перцентиля
```

Такой подход хорошо подходит для ежегодного мониторинга, потому что автоматически адаптируется к масштабу нового массива данных.

## 3. Непокрытая нагрузка

**Смысл:** оценивает не просто количество непокрытых направлений, а вес этих направлений.

**Формула:**

```text
Непокрытая нагрузка = сумма уровней нагрузки по активным направлениям, не включенным в СВК
```

**Пример:**

- закупки: уровень 3, не покрыто;
- имущество: уровень 2, покрыто;
- проекты: уровень 4, не покрыто.

```text
Непокрытая нагрузка = 3 + 4 = 7
```

## 4. Форма СВК

Форма СВК кодируется по шкале возрастания организационной силы:

| Уровень | Форма |
|---:|---|
| 0 | Форма не определена |
| 1 | Уполномоченное должностное лицо |
| 2 | Временный коллегиальный орган / комиссия |
| 3 | Постоянно действующий коллегиальный орган |
| 4 | Уполномоченное структурное подразделение |

Если организация отметила несколько форм, берется максимальный уровень.

## 5. Расчетно требуемая форма СВК

**Смысл:** какая минимальная форма СВК ожидается при имеющемся объеме деятельности.

Базово:

```text
Требуемая форма = максимальный уровень нагрузки по активным направлениям
```

Дополнительно применяется поправка на комплексность:

```text
если 3 или более направления имеют нагрузку 2+, требуемая форма повышается на 1 уровень
```

Максимум — 4.

## 6. Разрыв формы СВК

**Смысл:** показывает, соответствует ли фактическая форма внутреннего контроля объему деятельности.

**Формула:**

```text
Разрыв = фактический уровень формы СВК − расчетно требуемый уровень формы СВК
```

**Интерпретация:**

| Значение | Интерпретация |
|---:|---|
| +1 и выше | форма сильнее минимально необходимой |
| 0 | форма соответствует нагрузке |
| -1 | умеренный дефицит формы |
| -2 | существенный дефицит формы |
| -3 / -4 | критический дефицит формы |

## 7. Группы риска

| Группа | Условие |
|---|---|
| A. Сбалансированная СВК | активные направления покрыты, форма соответствует нагрузке |
| B. Недопокрытие активных направлений | есть активность, но одно или несколько направлений не включены в СВК |
| C. Слабая форма СВК | направления могут быть покрыты, но форма ниже расчетно необходимой |
| D. Двойной риск | есть и непокрытие направлений, и слабая форма |
| E. Отчет не заполнен / данные требуют проверки | отчет не заполнен или есть критичные признаки низкого качества данных |

## 8. Флаги качества данных

Проект автоматически отмечает потенциальные противоречия:

- СВК указана как организованная, но отсутствуют базовые элементы;
- устраненных нарушений больше, чем выявленных;
- высокая нагрузка при отсутствующей или минимальной форме СВК;
- высокая нагрузка по направлению, но направление не покрыто СВК.

Эти флаги не являются окончательным выводом о нарушении. Это список анкет для проверки.


## 9. Общая статистика отчетности и зрелости СВК

Этот блок добавлен для воспроизведения общей аналитики из первичного обзора отчета. Он считается автоматически при каждой замене годового файла.

### 9.1. Статус отчетности

**Метрики:**

- количество организаций в отчете;
- количество заполненных отчетов;
- доля заполненных отчетов;
- распределение по статусам отчета.

### 9.2. Базовые элементы СВК

Для каждого элемента считается количество организаций с ответом «Да», «Нет», пропусками и доля «Да» среди заполненных отчетов:

- СВК организован и осуществляется;
- раздел СВК в учетной политике;
- организация СВК утверждена ЛНА;
- определены полномочия ответственных лиц;
- утвержден план-график мероприятий СВК.

Дополнительно считается показатель **полного базового комплекта СВК**: учетная политика + ЛНА + полномочия + план-график.

### 9.3. Формы СВК

Считаются два представления:

1. сколько организаций отметили каждую форму СВК;
2. какая максимальная фактическая форма СВК определена у организации по шкале 0–4.

Это позволяет увидеть не только «самую сильную» форму, но и сочетание форм, если организация отметила несколько вариантов.

### 9.4. Рискоориентированный подход

Исходный текстовый ответ классифицируется в три основные группы:

- полноценная методика;
- фрагментарная / несистемная методика;
- методика отсутствует.

### 9.5. Внутренняя оценка эффективности СВК

Ответ классифицируется в группы:

- регулярная оценка;
- нерегулярная оценка / результаты не оформляются;
- оценка не проводится.

### 9.6. Совершенствование СВК

Ответ классифицируется в группы:

- документированные мероприятия реализуются;
- мероприятия планируются или выполняются частично;
- мероприятия не планируются / не проводятся.

### 9.7. Нарушения и меры реагирования

Считаются:

- общее количество выявленных нарушений;
- количество и доля нарушений, устраненных в срок;
- количество и доля нарушений, устраненных с нарушением срока;
- остаток без отраженного устранения;
- планы устранения;
- решения по внесению изменений в ЛНА;
- решения о дисциплинарной ответственности;
- материалы в правоохранительные / контрольные органы;
- материалы в суд;
- число строк с арифметическим противоречием: устранено больше, чем выявлено.

### 9.8. Нормированные показатели

Для сопоставимости крупных и малых организаций рассчитываются:

- нарушений на 100 среднесписочных сотрудников;
- нарушений на 1 млрд руб. кассовых выплат;
- нарушений на 1 000 фактов хозяйственной жизни;
- закупок на 100 сотрудников;
- доля объема закупок в кассовых выплатах.
