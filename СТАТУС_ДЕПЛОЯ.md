1. Зайдите на **https://streamlit.io/cloud**
2. Войдите через **GitHub** аккаунт

   - **Repository:** `svk-analytics-streamlit`

   - **Main file path:** `app.py`

5. (Опционально) **Advanced settings:**

## 💡 Альтернативные варианты деплоя:

### Docker (локально или на сервере):
```bash
docker build -t svk-analytics-streamlit .
docker run -p 8501:8501 svk-analytics-streamlit
# Открыть http://localhost:8501
```

### Локальный запуск:
```bash
git clone https://github.com/ваш-username/svk-analytics-streamlit.git
cd svk-analytics-streamlit
python -m venv .venv
source .venv/bin/activate  # macOS/Linux
pip install -r requirements.txt
streamlit run app.py
```
