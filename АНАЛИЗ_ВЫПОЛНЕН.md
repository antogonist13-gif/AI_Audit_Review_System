## 🔧 ЧТО БЫЛО ИСПРАВЛЕНО

### Проблема
Функция `yes_flag()` в `src/svk_analytics/cleaning.py` не работала с Pandas 3.0 (ArrowStringArray backend).

### Решение
Добавлена проверка типа данных и конвертация:
```python
if s.dtype == 'object' or hasattr(s.dtype, 'pyarrow_dtype'):
    s = s.astype(str)
```

Заменен `.str.match()` на `.str.startswith()` для корректной работы с pyarrow.
